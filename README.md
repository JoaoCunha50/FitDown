## Building

### Dependencies

 - JAVA 
 - JDK (Version 8 or above)
 - Bash (build-time)

### Building

A release build is achieved with:

``` bash
$ make
```

To remove files generated by a build, run:
```bash
$ make clean
```
You can build and run the App with:

```bash
$ make run
```
